# Shree Choice Playworld — Catalog Automation Pipeline

AI-powered catalog automation system for a retail business 
managing 5,000+ product inventory.

## What it does
- Parses bulk distributor PDFs using Gemini API
- Extracts structured pricing and image payloads automatically
- Syncs 5,000+ product records to primary database
- Preview staging dashboard for safe data validation before sync

## Problem it solves
Manual catalog entry for 5,000+ items was taking days.
This pipeline reduced it to minutes with zero manual data entry.

## Stack
`Python` `Gemini API` `FastAPI` `PDF Parsing` `Docker`
